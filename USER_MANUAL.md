# User Manual - Event Manager

## 1. What This Application Does

Event Manager is a web application for creating events, registering participants, assigning volunteers, and tracking attendance with QR codes. It provides separate access for participants, volunteers, organizers, and administrators.

## 2. Who Can Use It

- **Participants** can browse events, register, and view their attendance history.
- **Volunteers** can scan QR codes and help manage attendance.
- **Organizers** can create and manage events, volunteers, and registrations.
- **Administrators** can manage users, events, and the overall system.

## 3. How to Access the Application

1. Open the website in your browser.
2. Use the home page to navigate to login or registration.
3. If you already have an account, sign in with your username and password.
4. If you do not have an account, create one using the registration form.

## 4. First-Time User Steps

### For Participants

1. Open the registration page.
2. Enter your full name, username, email, phone number, date of birth, and password.
3. Submit the form.
4. Log in with your new account.
5. Open the dashboard to view events and your QR code.

### For Organizers and Volunteers

1. Sign in with your assigned account.
2. Open the dashboard to view your assigned events.
3. Use the event pages to manage tasks such as event setup, volunteer work, or attendance scanning.

## 5. Main Pages and What They Do

### Home Page

The home page is the starting point of the application. It gives access to login, registration, and the main application sections.

### Dashboard

The dashboard gives a summary of your account activity.

- Participants see their registered events, attendance count, and QR code.
- Organizers see their managed events and registration activity.
- Volunteers see the events they are assigned to support.

### Events Page

Use the events page to:

- View published events
- Create events if you are an organizer
- Edit or delete your own events
- Assign or remove volunteers

### Registrations Page

Use the registrations page to:

- View your own registrations as a participant
- Review all event registrations if you are an organizer
- Approve or reject registrations for your events
- Register or unregister for events

### Attendance Page

Use the attendance page to:

- View attendance records
- Open the QR scanner for check-in
- Check in approved attendees
- Check out attendees when needed

### Admin Panel

Use the admin panel to manage system-level data such as users, events, and approvals. This section is available only to authorized users.

## 6. Common Actions

### Register for an Event

1. Log in as a participant.
2. Open the events page.
3. Select a published event.
4. Click the register option.
5. Wait for approval if the event uses approval-based registration.

### View Your Registrations

1. Log in.
2. Open the registrations page.
3. Review your pending, approved, or rejected registrations.

### Check In Using QR Code

1. Make sure your registration is approved.
2. Go to the event location.
3. Show your QR code to the volunteer or organizer.
4. The volunteer scans the QR code.
5. If the code is valid, your attendance is recorded.

### Create an Event

1. Log in as an organizer.
2. Open the events page.
3. Click create event.
4. Fill in the event details.
5. Save the event as draft or publish it when ready.

### Manage Volunteers

1. Open your event details as an organizer.
2. Open the volunteer management section.
3. Add or remove volunteer users as needed.

## 7. QR Code Attendance

The QR code is used to securely identify a user during event check-in.

- Each user gets a unique QR token.
- The token is verified before attendance is recorded.
- Duplicate check-ins are blocked.
- If a QR code is invalid or tampered with, the system rejects it.

## 8. What You Should See After Login

- A personalized dashboard
- Navigation links based on your role
- Event and registration information relevant to your account
- Attendance or management options if your role allows them

## 9. Common Problems and Fixes

### I Cannot Log In

- Make sure your username and password are correct.
- Check whether your account was created successfully.
- Ask an administrator if your role or account status is not correct.

### I Cannot Register for an Event

- The event may already be full.
- The registration deadline may have passed.
- You may already be registered for that event.

### My QR Code Does Not Work

- Make sure you are using your own QR code.
- Check whether your registration has been approved.
- Ask the organizer or volunteer to scan it again.

### I Do Not See the Option I Need

- Some pages are restricted by role.
- Log in with the correct account type.
- Contact an organizer or administrator if needed.

## 10. Logging Out

When you are finished, use the logout option to safely end your session.

## 11. Contact and Support

If something does not work as expected, contact the project administrator or the person managing the event system.
